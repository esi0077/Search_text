# Search your file and find text inside it 
<hr>

#### How to use❓

open cmd and type this:

```bash
python main.py
```

the app going to pop up!

#### ⚠️ note 

you need to have your .text file inside the same folder as script


#### 📂 Folder Structure 

- `Folder/`
  - `whatever.txt`
  - `main.py`
